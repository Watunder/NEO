Class CL3D.Animator

An animator can be attached to a scene node and animates it.

Class Overview
CL3D.Animator()
An animator animates a scene node. It can animate position, rotation, material, and so on. A scene node animator is able to animate a SceneNode in a very simple way: It may change its position, rotation, scale and/or material. There are lots of animators to choose from. You can create scene node animators and attach them to a scene node using SceneNode.addAnimator().
Note that this class is only the base class of all Animators, it doesn't do anything itself. See AnimatorCameraFPS for a concrete Animator example.
